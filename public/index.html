📊 Sistema de Gestión de Inventario

Manage your products and clients efficiently

📦 Productos 👥 Clientes 🔗 API

📦 Gestión de Productos

Accede a los endpoints para gestionar productos:

GET /productos - Obtener todos los productos
GET /productos/:id - Obtener un producto específico
POST /productos - Crear un nuevo producto
DELETE /productos/:id - Eliminar un producto

Ejemplo de datos: {"nombre": "Laptop", "precio": 1200, "cantidad": 10, "categoria": "Electrónica"}

👥 Gestión de Clientes

Accede a los endpoints para gestionar clientes:

GET /clientes - Obtener todos los clientes
GET /clientes/:id - Obtener un cliente específico
POST /clientes - Crear un nuevo cliente
DELETE /clientes/:id - Eliminar un cliente

Ejemplo de datos: {"nombre": "Juan Pérez", "email": "juan@example.com", "telefono": "123456789"}
